Connecting with Perforce

SciTech Software uses the commercial Perforce version management software to maintain it's public Open Source software repository. Perforce Software has graciously allowed Open Source software projects to be able to use a free server license for that source code. This section details how you can use Perforce to keep up to date with the absolute latest versions of the SciTech SNAP SDK.
