REF2D_driver

Prototype In

snap/graphics.h

Description

Structure for the 2d referster rasteriser, which is returned by the REF2D_loadDriver function. This structure also contains all the function pointers used to communicate with the 2d reference rasteriser library.

Note:    Be sure to fill in the dwSize member of this structure when you call GA_queryFunctions to the correct size of the structure at compile time!
