Buffer Manager Functions

These functions are members of the GA_bufferFuncs group. The functions provide support for allocating, freeing, copying and drawing on offscreen video memory buffers. The buffer manager functions should always be used to manage offscreen surfaces when possible, for maximum performance and compatibility.
