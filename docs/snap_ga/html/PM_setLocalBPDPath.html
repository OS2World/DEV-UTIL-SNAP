PM_setLocalBPDPath

Function to override the SNAP BPD driver path

Declaration

void PMAPI PM_setLocalBPDPath(
    const char *path)

Prototype In

pmapi.h

Parameters

path

Local path to the SciTech SNAP BPD driver files.

 

Description

This function is used by the application program to override the location of the SciTech SNAP driver files that are loaded. Normally the loader code will look in the system SciTech SNAP directories first, then in the 'drivers' directory relative to the current working directory, and finally relative to the MGL_ROOT environment variable. By default the local BPD path is always set to the current directory if not initialised.
