PM_sleep

Sleep for the specified number of milliseconds.

Declaration

void PMAPI PM_sleep(
    ulong milliseconds)

Prototype In

pmapi.h

Parameters

milliseconds

Number of milliseconds to sleep for

 

Description

This function is used to pause the current process and put it to sleep for the specified number of milliseconds.
