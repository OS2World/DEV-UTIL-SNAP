DPMSsetState

Sets the requested DPMS Power Management state.

Declaration

void NAPI GA_DPMSFuncs::DPMSsetState(
    N_int32 state)

Prototype In

snap/ddc.h

Parameters

state

New DPMS state to set

 

Description

This function sets the requested Power Management state. The list of valid states that can be set is defined in the DDC_DPMSStatesType enumeration.
