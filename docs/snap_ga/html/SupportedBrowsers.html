Supported Browsers

This help system requires a web browser with support for Java and Javascript.  Browsers that are known to work are Internet Explorer version 5.0 and above, Netscape version 7.0 and above, and Mozilla version 1.0 and above. Other browsers, such as Netscape 4.x, may not have the required Javascript support. If your browser is not capable of supporting Java, you can view the documentation with no Java table of contents browser here:

http://www.scitechsoft.com/docs/snap_ga/notoc/index.html

If you have a supported browser but you cannot see the table of contents pane on the left side of the web page, you will need to install the Java Runtime Environment for your brower. Microsoft Windows Update contains support for installing the Microsoft Java Virtual Machine for use with Internet Explorer. Sun Microsystems provides Java support for Windows, Linux, and Solaris.  IBM DeveloperWorks provides Java support for AIX, Linux, OS/2, z/OS, and Windows.
